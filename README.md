# Connect4: Nexus

Современная версия "Четыре в ряд" в формате киберпанк-платформы: режимы для соло и PvP, mutators, прокачка, магазин и кастомизация интерфейса.

## Для кого

- Для игроков, которым нужен быстрый, зрелищный и переигрываемый strategy-casual опыт.
- Для друзей: можно играть локально на одном ПК и тестировать online beta через WebSocket.

## Почему это ценно

- Базовая механика Connect4 остается понятной, но глубина растет за счет режимов и модификаторов.
- Один продукт закрывает сразу несколько сценариев: тренировка с AI, локальные дуэли, соревновательный стиль.
- Есть фундамент для масштабирования: ranked, турниры, онлайн-инфраструктура, монетизация.

## Быстрый старт

```bash
npm install
npm run dev
```

Приложение откроется на `http://localhost:5173/`.
